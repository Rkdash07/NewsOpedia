Project Description

NewsOpedia is a responsive and user-friendly news platform built to fetch and display the latest headlines from reliable news APIs.
It offers:

Clean and minimal user interface

Multiple news categories

Search & filter functionality

Error handling & loader states

Dark/Light mode (if available)

Fast build & optimized deployment

This project demonstrates modern web development practices including component-based architecture, API integration, and responsive UI.
